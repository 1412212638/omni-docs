# 可接受使用政策

生效日期：2026 年 3 月 25 日

本《可接受使用政策》补充《[用户协议](/zh/legal/terms)》，适用于您通过 OmniRouters 网站、控制台、API、文档和相关工具开展的全部活动。

## 1. 基本原则

您仅可在合法、正当、已获授权且符合本政策的前提下使用 OmniRouters。您不得通过平台实施、协助、放大或规避任何违法、侵权、欺诈或滥用行为。

## 2. 禁止行为

### 2.1 违法与侵权内容

您不得利用 OmniRouters 生成、存储、处理、传播或协助传播以下内容：

- 违法内容、侵权内容、虚假宣传、诈骗信息或侵犯第三方合法权益的材料
- 恶意诽谤、骚扰、威胁、仇恨、暴力煽动或明显有害内容
- 恶意冒充、欺骗性身份伪装、误导性深度伪造或非授权身份仿冒

### 2.2 安全与平台滥用

您不得：

- 绕过认证、速率限制、权限控制或其他安全措施
- 扫描、探测、攻击、压测、抓取、镜像或干扰平台、模型路由或相关基础设施
- 传播恶意代码、木马、勒索软件、僵尸网络控制代码或其他恶意程序
- 进行撞库、窃取凭证、批量试错、盗刷或其他账户攻击行为

### 2.3 欺诈与不当业务行为

您不得利用 OmniRouters 从事：

- 钓鱼、冒充客服、支付欺诈、营销诈骗、刷量、垃圾信息传播
- 隐藏或伪造真实调用来源、计费责任或业务主体
- 违反适用支付规则、出口管制、制裁要求或行业监管义务的活动

### 2.4 高风险与受限制场景

对于可能影响个人重大权益的场景，您应进行独立评估并保持人工复核。未经充分保障，您不得将 OmniRouters 输出直接作为以下事项的唯一依据：

- 就业、解雇、绩效考核、教育录取
- 贷款、信用、保险、住房、公共服务资格
- 医疗诊断、治疗决策、法律意见或执法判断
- 身份验证、实时监控、敏感画像或其他高风险自动化决策

### 2.5 知识产权与数据合规

您不得：

- 上传、传输或处理您无权使用的数据、代码、文档、模型或素材
- 在无适当法律基础、授权或通知的情况下处理第三方个人信息
- 要求 OmniRouters 规避平台限制、第三方规则或监管义务

## 3. 执行措施

如我们认为您违反本政策或存在明显风险，我们可视情况采取以下措施：

- 警告、限制功能、限制速率或要求补充说明
- 暂停、终止账户、撤销 API Key 或阻断相关请求
- 冻结或拒绝退款申请
- 保存并向有关方披露必要记录，以满足法律、监管、司法或安全要求
