## Best for

- text-to-speech
- voice selection
- speed and format control
- timestamp and emotion metadata

## Default routing

- standard TTS -> `/v1/audio/speech`
- Gemini-compatible speech -> `/v1beta/models/gemini-2.5-flash-preview-tts:generateContent`
